# PS8: AI for Industrial Knowledge Intelligence

A unified AI-powered knowledge companion that instantly answers equipment-related questions by searching across all uploaded documents simultaneously, providing context-aware answers with exact citations, identifying patterns, predicting failures, and recommending preventive actions.

## Project Structure
- `backend/`: FastAPI application, endpoints, RAG logic, and ingestion pipelines.
- `frontend/`: React frontend for user interaction and dashboard.
- `data/`: Storage for sample documents and user uploads.
- `docs/`: Additional project documentation.
